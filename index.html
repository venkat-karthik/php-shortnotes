⚡ Topic 1

What is PHP?

Discover the server-side scripting language that powers over 77% of the web. PHP transforms static HTML into dynamic, interactive experiences.

🚀 The PHP Advantage

PHP is a server-side scripting language that generates dynamic content before sending it to the browser. Unlike static HTML, PHP can:

  • Respond to different users with personalized content
  • Connect to databases and retrieve real-time data
  • Process form submissions securely on the server
  • Create sessions and manage user authentication

How PHP Works

1
User visits localhost/index.php in their browser
↓
2
Request reaches the Apache web server
↓
3
PHP engine processes the .php file and executes PHP code
↓
4
Pure HTML is generated and sent back to the browser
💡 Pro Tip: The browser never sees PHP code — it only receives the final HTML output. This is what makes PHP secure for backend logic.

📝 Basic PHP Syntax

<?php
  echo "Hello, World!";
?>

Installation & XAMPP

Get your local development environment up and running in minutes.

Why XAMPP?

PHP cannot run standalone. You need three components working together:

Component Purpose What It Does
Apache Web Server Receives requests and serves files to browsers
PHP Interpreter Executes PHP code on the server
MySQL Database Stores and manages data persistently

📂 Project Structure

xampp/
 └── htdocs/              ← All your projects go here
      ├── project1/
      │    └── index.php
      └── project2/
           └── index.php

Access in browser: localhost/project1

⚠️ Common Mistake: Never place your PHP files on the Desktop or Documents. They MUST be inside the htdocs folder or PHP won't execute them.

📦 Topic 6

Understanding Variables

Variables are containers that store and manage data in your programs.

What is a Variable?

Think of a variable as a labeled box. You put something inside it, give it a name, and retrieve it later. In PHP, all variables start with a $ sign.

💾 Creating and Using Variables

<?php
$name = "John";
$age = 20;
$cgpa = 8.5;
$isStudent = true;

echo $name;    // Output: John
echo $age;     // Output: 20
?>

📋 Variable Rules

Valid Invalid Reason
$name $1name Cannot start with a number
$_name $name% No special characters except underscore
$name1 name Must start with $
⚠️ Case Sensitive: $name and $Name are different variables in PHP.

🔄 Changing Variables

<?php
$name = "John";
echo $name;      // Output: John

$name = "Mike";
echo $name;      // Output: Mike
?>
🔗 Topic 13

URL Parameters

Pass data through URLs using query strings.

URL parameters use the ?key=value syntax to pass data:

http://localhost/test.php?name=John&age=20&city=NYC

Accessing Parameters in PHP

<?php
echo $_GET["name"];    // John
echo $_GET["age"];     // 20
echo $_GET["city"];    // NYC
?>
Always use $_GET["key"] with quotes around the key name, not $_GET[key].

📊 Topic 15

Arrays Explained

Store multiple values in a single variable with arrays.

Creating Arrays

<?php
$friends = array("John", "Mike", "Tom", "Sara");
//           [0]      [1]      [2]    [3]

echo $friends[0];  // John
echo $friends[2];  // Tom
?>

Modifying Arrays

<?php
$friends[1] = "David";    // Replace Mike
$friends[4] = "Chris";    // Add new element
count($friends);          // Returns 5
?>
⚠️ Array indexes start at 0, not 1. Accessing $friends[99] produces a warning if it doesn't exist.

Handling Checkboxes

Process multiple checkbox selections from HTML forms.

HTML Form with Multiple Checkboxes

<form method="post">
  <input type="checkbox" name="fruits[]" value="Apple">
  <input type="checkbox" name="fruits[]" value="Banana">
  <input type="checkbox" name="fruits[]" value="Mango">
  <input type="submit">
</form>

PHP Processing

<?php
if(isset($_POST["fruits"])) {
    $fruits = $_POST["fruits"];
    print_r($fruits);
}
?>
Always wrap with isset() because if no boxes are checked, the key won't exist in $_POST.
🔑 Topic 17

Associative Arrays

Use named keys instead of numbers for more readable code.

Creating Associative Arrays

<?php
$grades = array(
    "John" => "A",
    "Mike" => "B",
    "Tom" => "C"
);

echo $grades["John"];   // A
$grades["John"] = "A+"; // Update
?>
Use string keys for natural labels. This is much clearer than numeric indexes for data like names, cities, or prices.

Functions

Variable Scope

<?php
$x = 10;  // Global variable

function test() {
    global $x;  // Access the global $x

}

test();
?>
Variables inside a function are local by default. Use global keyword to access variables from outside the function.

Building a Calculator

Combine forms, variables, and conditionals into a working calculator.

HTML Form

<form method="get">
  <input type="number" name="num1">
  <input type="text" name="op" placeholder="+">
  <input type="number" name="num2">
  <input type="submit" value="Calculate">
</form>

PHP Logic

<?php
if(isset($_GET["num1"])) {
    $num1 = $_GET["num1"];
    $num2 = $_GET["num2"];
    $op = $_GET["op"];
    
    if($op == "+") echo $num1 + $num2;
    elseif($op == "-") echo $num1 - $num2;
    elseif($op == "*") echo $num1 * $num2;
    elseif($op == "/") echo $num1 / $num2;
}
?>
🔀 Topic 23

Switch Statements

A cleaner way to handle many conditions.

Switch vs If/Else

<?php
$grade = "A";

switch($grade) {
    case "A":
        echo "Excellent!";
        break;
    case "B":
        echo "Good!";
        break;
    case "C":
        echo "Average";
        break;
    default:
        echo "Invalid";
}
?>
⚠️ Always use break; to prevent "fall-through". Without it, execution continues into the next case.
🔄 Topic 24

While Loops

Repeat code while a condition is true.

While Loop Syntax

<?php
$i = 1;
while($i <= 5) {
    echo $i;
    $i++;
}
// Output: 12345
?>
⚠️ Infinite Loop: Forgetting to increment $i++ creates an infinite loop and crashes your server.
📈 Topic 25

For Loops

The most powerful and commonly used loop type.

For Loop Syntax

<?php
for($i = 1; $i <= 5; $i++) {
    echo $i;
}
// Output: 12345

// Multiplication table
for($i = 1; $i <= 10; $i++) {
    echo "5 × $i = " . (5 * $i) . "<br>";
}
?>

Classes & Objects

<?php

🔨 Topic 30

Constructors

Automatic initialization when creating objects.

Constructor Method

<?php
class Student {
    public $name;
    public $age;
    
    function __construct($name, $age) {
        $this->name = $name;
        $this->age = $age;
    }
}

$s1 = new Student("John", 20);
echo $s1->name;  // John (set automatically)
?>

🔐 Topic 32

Getters & Setters

Control access to object properties with validation.

Protected Properties with Getters/Setters

<?php
class Student {
    private $cgpa;
    
    function setCgpa($value) {
        if($value >= 0 && $value <= 10) {
            $this->cgpa = $value;
        }
    }
    
    function getCgpa() {
        return $this->cgpa;
    }
}

$s1 = new Student();
$s1->setCgpa(8.5);    // ✅ Accepted
$s1->setCgpa(500);    // ❌ Rejected
?>
This is called Encapsulation — protecting internal data and providing controlled access points.
🧬 Topic 33

Inheritance

Reuse code by inheriting from parent classes.

Parent & Child Classes

<?php
class Person {
    public $name;
    public $age;
}

class Student extends Person {
    public $cgpa;
}

$s1 = new Student();
$s1->name = "John";   // Inherited
$s1->age = 20;        // Inherited
$s1->cgpa = 8.5;      // Student-specific
?>

↓
2

Inheritance eliminates code duplication. Fix a bug once in the parent class and all child classes inherit the fix automatically.
